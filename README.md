## Как использовать бота

Для использование бота, необходимо настроить конфигурации в файле `config.py`.

Первоначально подставляем токены ботов:
```
token_telegram = '' /сюда подставляем токен созданного вами бота в телеграме
token_vk = '' /сюда подставляем токен созданного вами бота в вк
```
Словарь `correcr_id` испльзуется для чтения ботом только разрешенных пользователей в беседах.
```
correct_id = {
    'telegram_id' : 'teacher_name', / Указываете телеграм id и имя преподавателя
    'telegram_id' : 'teacher_name' 
}
```
Словарь `groups` используется для пересылки сообщений в беседу,если номер группы был в тексте сообщения.
```
groups = {
    '352' : 1, / Ключ- номер группы , Значение - id группы для бота
    '331' : 2,
    '351' : 3
}
```
> Важно помнить,что при создании бота в ВК , id бесед, в которые его добавляют начинается с 1>
